  timeout: Timeout for each individual test case. This setting specifies the maximum time allowed for each test case to complete. If a test case exceeds this timeout, it will be considered failed due to timeout.

pageTimeout: Timeout for each page action. This setting specifies the maximum time allowed for each page action (such as navigation, click, fill, etc.) to complete. If a page action exceeds this timeout, it will be considered failed due to timeout.

contextOptions.timeout: Timeout for each browser context. This setting specifies the maximum time allowed for each browser context to be created. If the creation of a browser context exceeds this timeout, it will be considered failed due to timeout.

defaultBrowserOptions.timeout: Timeout for browser launch. This setting specifies the maximum time allowed for the browser to be launched. If the browser launch exceeds this timeout, it will be considered failed due to timeout.

defaultBrowserOptions.argsTimeout: Timeout for browser arguments. This setting specifies the maximum time allowed for passing arguments to the browser process. If passing arguments to the browser process exceeds this timeout, it will be considered failed due to timeout.
